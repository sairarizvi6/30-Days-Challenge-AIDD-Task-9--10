# Plot Consistency Checker

## Purpose
Ensures plot elements, character actions, and timeline remain consistent throughout the book.

## Input
- Manuscript chapters or scenes
- Character profiles
- Timeline of events

## Process
1. Track character locations and actions across chapters
2. Verify timeline consistency
3. Check for plot holes or contradictions
4. Validate character behavior against established traits
5. Ensure foreshadowing pays off
6. Verify world-building rules are maintained

## Output
Detailed report with:
- Identified inconsistencies
- Timeline conflicts
- Character behavior issues
- Plot holes
- Suggestions for fixes

## Example Usage
```
Check chapters 1-10 for plot consistency, focusing on the main character's timeline and the magic system rules.
```

## Best Practices
- Maintain a master timeline
- Track character knowledge and capabilities
- Document world-building rules
- Cross-reference major plot points
