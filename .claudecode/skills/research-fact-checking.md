# Research and Fact-Checking Tool

## Purpose
Researches and verifies factual information for non-fiction or realistic fiction elements.

## Input
- Topic or claim to verify
- Context (historical period, location, technical subject)
- Required depth of research

## Process
1. Identify verifiable facts in the content
2. Research using reliable sources
3. Cross-reference multiple sources
4. Flag unsupported claims
5. Provide citations and references
6. Suggest accurate alternatives

## Output
- Fact-check report
- Source citations
- Accuracy rating
- Corrections and suggestions
- Additional context

## Example Usage
```
Fact-check historical details about Victorian London in chapter 3, including social customs and technology.
```

## Best Practices
- Use primary sources when possible
- Cross-verify from multiple sources
- Consider historical context
- Document all sources
