# Chapter Outline Generator

## Purpose
Generates detailed chapter outlines for books based on genre, themes, and target audience.

## Input
- Book genre
- Target audience
- Main themes
- Desired chapter count

## Process
1. Analyze the genre conventions and reader expectations
2. Structure the narrative arc across chapters
3. Define key plot points and character development moments
4. Ensure pacing and balance
5. Create chapter summaries with key events

## Output
A structured outline with:
- Chapter titles
- Key events per chapter
- Character arcs
- Plot progression
- Word count estimates

## Example Usage
```
Generate a 15-chapter outline for a young adult fantasy novel about a teen discovering magical powers, themes of identity and friendship.
```

## Best Practices
- Consider the three-act structure
- Balance action and character development
- Plan cliffhangers for engagement
- Ensure logical plot progression
