# Character Development Assistant

## Purpose
Creates deep, multi-dimensional characters with consistent personalities, backgrounds, and arcs.

## Input
- Character role (protagonist, antagonist, supporting)
- Story genre and setting
- Character's purpose in the narrative

## Process
1. Generate character biography and background
2. Define personality traits using psychological frameworks
3. Create character motivations and goals
4. Develop character flaws and strengths
5. Map character arc throughout the story
6. Define relationships with other characters

## Output
Complete character profile including:
- Physical description
- Personality traits
- Background/history
- Motivations and goals
- Character arc
- Dialogue style
- Strengths and weaknesses

## Example Usage
```
Create a protagonist for a mystery thriller: a 35-year-old detective with a traumatic past.
```

## Best Practices
- Make characters three-dimensional
- Give them contradictions and flaws
- Ensure motivations drive the plot
- Create distinct voices for each character
